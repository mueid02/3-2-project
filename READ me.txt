The 'finalproject' folder contains source code for the website with functionality implemendted using PHP and a MySQL database. Please, make sure you have XAMPP installed on your computer.
To run this website in your computer, you will need to, 
 - create a database and import the database project.sql file
 - change the database credentials inside the connect.php with your own database credentials.
 - and if needed, then you have to modify the BASE_URL variable inside the path.php.
